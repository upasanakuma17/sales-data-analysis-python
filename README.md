# Sales Data Analysis using Python

## 📌 Project Overview
This project analyzes sales data to uncover customer behavior and improve business decision-making.

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn

## 🔍 Key Steps
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Customer segmentation based on demographics
- Product category performance analysis

## 📊 Insights
- Identified top-performing product categories
- Discovered key customer segments
- Suggested improvements for inventory planning

## 🚀 Outcome
Improved understanding of customer behavior and helped in making data-driven business decisions.
